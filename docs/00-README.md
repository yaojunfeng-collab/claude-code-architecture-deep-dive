# Claude Code 2.1.88 源码深度剖析 — 技术分享系列

> 面向高级工程师的 10 讲系统性技术分享
> 基于 Claude Code 2.1.88 源码阅读，每讲围绕一个核心模块展开

---

## 系列总览

这个系列不是 API 文档，也不是使用教程。它回答一个核心问题：

**Anthropic 的工程师是如何把一个"能聊天的 LLM"变成一个"能写代码的 Agent"的？**

每一讲都围绕一个模块，拆解其设计决策、工程权衡和实现细节。适合：
- 正在构建 AI Agent / Coding Assistant 产品的工程师
- 想理解 Claude Code 为什么这么设计的重度用户
- 对大型 LLM 应用架构感兴趣的技术负责人

## 每讲的结构

每一讲都按统一框架组织：

```
[开场] 一个问题引入 — 为什么这个模块存在？解决什么痛点？
[全景] 架构鸟瞰图 — 30 秒理解整体结构
[深入] 逐层拆解 — 源码级的设计决策分析
[设计] 为什么这么做 — tradeoff 分析、与竞品对比
[陷阱] 常见误区 — 容易理解错的地方
[关联] 跨模块联系 — 和其他讲的关系
[收获] 核心 takeaway — 3-5 个带走的洞察
[思考] 延伸问题 — 如果是你，你会怎么设计？
```
